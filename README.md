# Garden Dreams

## Description
Garden Dreams est un site web vitrine dédié à une boutique de fleurs spécialisée dans les roses et bouquets artisanaux. 
Réalisé en HTML5 et CSS3, le projet met en œuvre une structure sémantique claire, une navigation interne, 
des sections organisées, un tableau d’offres et une mise en page moderne et responsive.

## Technologies utilisées
- HTML5
- CSS3
